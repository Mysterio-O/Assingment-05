Question-1: What are the different ways to select an element in the DOM?

There are so many ways to select an element in the DOM I guess. But I know couple of them. First we should pick the element by using a javaScript function "document". Then we have to use another functions called "getElementById()", "getElementsByClassName()", "getElementByTagName()", "querySelectorAll()", "querySelector()". These functions help us to find an element from the document by their id, class, tag names. Also use can select them with query-selectors, in that case querySelectorAll() finds all those elements with that name or names. And querySelector() only selects the first element.
The whole scenario something looks like this : document.getElementById('Id name'); , document.getElementsByClassName('class name'); , document.getElementByTagName('tag name'); ,document.querySelectorAll('class name/names'); , document.querySelector('class name/names')

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerText and innerContent kind of same thing. Both show the text inside of an element. But one thing that innerText can do innerContent cant. innerText ignores hidden elements inside of the main element if there is any but in the same case, innerContent exposes the whole text inside of the element although it was a hidden text, but innerContent clearly ignores and show the whole text. And innerHTML returns the whole HTML structure. We can manipulate the inner-text with innerText or innerContent and also manipulate the inner-html structure with innerHTML tag. Example: ( element.innerText = 'new text'; / element.innerContent = 'new text'; and like this element.innerHTML = `HTML Structure`; we can add new HTML in the document. )

Question-3: What is event delegation in the DOM?

Event Delegation is an interesting thing in javaScript. it helps us to write less code and get more access. this concept is something like that, "you wanna tell something to your 5 friends, instead of telling them one by one you can call them together and tell whatever you want. By doing this, youre saving your time and you wont have to repeat those words." Event Delegation works exactly like this. Just have to call the parent element and there are bunch of options that we can access from there including elements and their styles and contents. We use 'target' attribute to do all these. This makes us to write less code.

Question-5: How do you create, add, and remove elements using JavaScript?

These things are the simplest things to do in javaScript. At first we need to select the section with javaScript selectors where we want to create, add or remove elements. Then we need to create an element with document.createElement('elementType'); . Then we will be able to catch the newly created element and write some html code like this: elementType.innerHTML = ` section i want to add using html. ` We can also use tailwind classes here. In that case we need to use cdn from tailwind docs. After completing the HTML structure we need to append that child in that selected element. We simply can do like this: selectedElement.appendChild(elementType);
We also can remove elements. There are couple of ways to do that. Either we can replace an empty string inside that element like this: const newContent = seletedElement.innerText = "";, or we can also use removeChild() function to remove entirely a child from parent container.
